## UI Components Required

### Confirm Paid Button
- Variant: default (accent color background)
- Size: sm
- Icon: Check icon (lucide)
- States:
  - Default: "Confirm Paid" label with check icon
  - Loading: disabled with spinner
  - Success: brief checkmark animation, then reset

### Recurring Transaction List
- Existing pattern in `PayScheduleList.tsx`
- Each row displays: amount, next due date, frequency
- Add confirm button to each row
- Overdue items: visual indicator (subtle red tint on date)
