Step 2 — After `app.New(cfg)` succeeds and BEFORE starting the server, register the
static middleware on `application.Echo`. API routes are already registered inside
`app.New()` → `handler.SetupRoutes()`, so static middleware added here is always AFTER
the API routes (correct ordering per research pitfall #1):

```go
// Serve the built React SPA from the embedded web/dist directory.
// HTML5: true forwards unknown paths to index.html for client-side routing.
// MUST be registered after SetupRoutes (already called inside app.New).
distFS, err := fs.Sub(webDist, "web/dist")
if err != nil {
    log.Fatalf("Failed to sub web/dist: %v", err)
}
application.Echo.Use(middleware.StaticWithConfig(middleware.StaticConfig{
    HTML5:      true,
    Root:       ".",
    Filesystem: http.FS(distFS),
}))
```

Note: `web/dist` does NOT exist until `npm run build` is run in `web/`. The embed
directive causes `go build` to fail if the directory is missing. To allow `go build` to
succeed before the frontend is built, create an empty placeholder:
```
web/dist/.gitkeep  (empty file, committed to git)
```
Create `web/dist/.gitkeep` as an empty file. This satisfies the embed pattern so the
Go build succeeds even before the first `npm run build`.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → Go API | HTTP requests from the SPA arrive as untrusted input |
| Embedded FS | `web/dist` is compiled into the binary at build time — content is trusted (build-time, not runtime) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01-01 | Tampering | Static file serving (`middleware.StaticWithConfig`) | accept | Content is embedded at build time from the repo; no runtime file writes; no path traversal possible through `http.FS(fs.Sub(...))` |
| T-05-01-02 | Information Disclosure | `/api/docs` endpoint (openapi.yaml) | accept | Personal tool on Tailscale; no sensitive secrets in the OpenAPI spec; endpoint documents public route shapes |
| T-05-01-03 | Elevation of Privilege | Route ordering — static catch-all before API routes | mitigate | API routes registered inside `app.New()` → `SetupRoutes()` which runs before `main.go` adds static middleware; test: `POST /api/check` must return JSON, not HTML |
</threat_model>
