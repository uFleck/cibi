## Implementation Notes

### Tailwind Breakpoints
- `lg:` = 1024px default Tailwind breakpoint
- Use `hidden lg:flex` to hide on mobile/tablet, show on desktop
- Use `lg:hidden` for opposite effect

### Body Scroll Lock (Mobile Drawer)
Use `document.body.style.overflow = 'hidden'` when drawer opens, restore on close. Prevents background scroll when drawer is open.

### AccountContext Pattern
All pages must read `selectedAccountId` from context and pass it to React Query keys:
```typescript
const { selectedAccountId } = useContext(AccountContext)
const currentAccountId = selectedAccountId || defaultAccountId
const { data } = useQuery({
  queryKey: ['resource', currentAccountId],
  queryFn: () => api.fetch(currentAccountId),
})
```

### Active Link Detection
Use `useLocation()` from TanStack Router to get current pathname and apply active styling:
```typescript
const location = useLocation()
const isActive = location.pathname === '/path'
```

### Icon Library
Use `lucide-react` (already installed):
- Home (Dashboard)
- Users (Accounts)
- FileText (Transactions)
- Settings (Settings)
- Menu (Hamburger)
- X (Close)
