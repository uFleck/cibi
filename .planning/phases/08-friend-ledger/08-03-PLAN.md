- Query: `useQuery({ queryKey: ['friends'], queryFn: listFriends })`

**Group Events section:**

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → /api/* | Authenticated CRUD — same-origin fetch, no CSRF needed for REST+JSON |
| Browser → /public/* | Public fetch by token — read-only, no user data modification possible |
| Clipboard API | Copy-link uses `navigator.clipboard.writeText` — browser prompts only if not HTTPS/localhost |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-03-01 | Information Disclosure | FriendPublicPage | accept | Token is the access credential; page is intentionally public per locked decision; no sensitive data beyond what the token holder should see |
| T-08-03-02 | Tampering | FriendsPage confirm button | mitigate | Confirm mutation calls authenticated /api endpoint; public page has no confirm UI — read-only enforced at UI layer and at API layer (public route has no confirm endpoint) |
| T-08-03-03 | Spoofing | Copy-link clipboard | accept | User initiates copy action; clipboard write requires user gesture in browsers — no silent exfiltration |
</threat_model>
