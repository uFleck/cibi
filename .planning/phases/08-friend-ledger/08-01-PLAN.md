Domain structs:
```go
type PeerDebt struct {
    ID                uuid.UUID
    FriendID          uuid.UUID
    Amount            int64     // cents; positive = friend owes user, negative = user owes friend
    Description       string
    Date              string    // RFC3339
    IsInstallment     bool
    TotalInstallments *int64    // nullable
    PaidInstallments  int64     // default 0
    Frequency         *string   // nullable, "monthly"
    AnchorDate        *string   // nullable, RFC3339
    IsConfirmed       bool
}

type PeerDebtBalance struct {
    FriendOwesUser int64  // sum of positive amounts for a friend (cents)
    UserOwesFriend int64  // abs(sum of negative amounts) for a friend (cents)
    Net            int64  // FriendOwesUser - UserOwesFriend (positive = friend net owes)
}

type GlobalPeerBalance struct {
    TotalOwedToUser int64  // sum of all positive amounts across all friends
    TotalUserOwes   int64  // abs(sum of all negative amounts)
    Net             int64
}
```

Interface `PeerDebtRepo`:
- `Insert(d PeerDebt) error`
- `GetByFriend(friendID uuid.UUID) ([]PeerDebt, error)`
- `GetAll() ([]PeerDebt, error)`
- `GetByID(id uuid.UUID) (PeerDebt, error)`
- `Update(id uuid.UUID, amount *int64, description *string, isConfirmed *bool, paidInstallments *int64) error`
- `DeleteByID(id uuid.UUID) error`
- `GetBalanceByFriend(friendID uuid.UUID) (PeerDebtBalance, error)` — single SQL query: `SELECT COALESCE(SUM(CASE WHEN amount > 0 THEN amount ELSE 0 END), 0), COALESCE(ABS(SUM(CASE WHEN amount < 0 THEN amount ELSE 0 END)), 0) FROM PeerDebt WHERE friend_id = ?`
- `GetGlobalBalance() (GlobalPeerBalance, error)` — same aggregation across all records

**Token generation helper** (add to `friend.go`, reuse in `group_event.go` via unexported package-level func in service package):

```go
// internal/service/token.go  — OR inline in friend.go as unexported func
func generatePublicToken() (string, error) {
    b := make([]byte, 16)
    if _, err := rand.Read(b); err != nil {
        return "", fmt.Errorf("generatePublicToken: %w", err)
    }
    return hex.EncodeToString(b), nil
}
```
